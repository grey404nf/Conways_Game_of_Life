# Conway's Game of Life
Conway's Game of Life in Go

## Usage
\[\<bottom or key>]: \<function>
- \[Left Bottom]: Add Cell(s)
- \[2]: 200*100 px
- \[4]: 400*200 px
- \[8]: 800*400 px
- \[P/p]: Pause
- \[S/s]: Start
- \[E/e]: Exit
- \[R/r]: Random State initialization
- \[N/n]: Empty State initialzation
- \[G/g]: Show "Gilder Gun"
- \[C/c]: Show "Centinal"
- \[F/f]: Show "Spacefiller"
- \[B/b]: Show "Breeder 1"
